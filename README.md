A collection of small Javascript (audio)visual pieces that I made between 2014 and 2017 (aged 17 to 19).

To link to this overview, you can use [this link](https://vvolhejn.github.io/etudes/).
Also check out my website [here](https://vvolhejn.com/).

## [Clockwise](https://vvolhejn.github.io/etudes/clockwise/) (Aug 2014)
Fractal-ish circles spinning clockwise.

## [Beige](https://vvolhejn.github.io/etudes/beige/) (Aug 2014)
A tiny, calming, interactive piece based on a recurring dream of mine. Try dragging or holding the mouse.

## [Depth](https://vvolhejn.github.io/etudes/depth/) (Oct 2014)
A simple [Sierpinski triangle](https://en.wikipedia.org/wiki/Sierpinski_triangle) zoom. It spins, too!

## [Pendulum](https://vvolhejn.github.io/etudes/pendulum/) (Oct 2014)
Simulates the famous pendulum wave effect.

## [Grow](https://vvolhejn.github.io/etudes/grow/) (Oct 2014)
Watch a randomly generated tree grow in real-time.

## [195](https://vvolhejn.github.io/etudes/195/) (Jan 2015)
Fractal mapping based on [xkcd 195](http://xkcd.com/195/).

## [Wolfram](https://vvolhejn.github.io/etudes/wolfram/) (Feb 2016)
Wolfram's cellular automata on a torus, with pretty colors. [Further instructions in the source.](wolfram/wolfram.js)

## [Web](https://vvolhejn.github.io/etudes/web/) (Dec 2016)
A minimalist web-like tessalation.
